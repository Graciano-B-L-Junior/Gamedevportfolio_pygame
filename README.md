# Game dev portfolio pygame

This is a portfolio of game development projects using Pygame.

## Projects

### 1. Flappy Bird

A recreation of the classic mobile game Flappy Bird. The player controls a bird that must navigate through a series of pipes without hitting them. The game features:

-   Simple one-button controls.
-   Procedurally generated pipes for endless gameplay.
-   Score tracking.
-   Game over screen with restart option.
